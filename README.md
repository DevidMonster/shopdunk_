# nest-project

## Run Client

Go to client directory

```javascript
    cd ./client
```

Install dependencies

```javascript
    npm i / npm install
```

Run project

```javascript
    npm run dev
```

## Run Server

Go to server directory

```javascript
    cd ./server
```

Install dependencies

```javascript
    npm i / npm install
```

Run project

```javascript
    npm run start:dev
```
